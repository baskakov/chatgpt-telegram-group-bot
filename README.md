# ChatGPT Telegram Group Bot

Telegram-бот на TypeScript, который отвечает на сообщения через ChatGPT и генерирует изображения через DALL·E.

---

## Возможности

- 💬 **ChatGPT-ответы** с сохранением контекста беседы (до 20 обменов)
- 🎨 **Генерация изображений** через DALL·E по триггерным словам
- 👥 **Работа в группах** — реагирует на упоминание `@username` или ответ на сообщение бота
- 🔒 **Whitelist** — доступ ограничен по ID пользователей и групп
- 🚪 **Авто-выход** из неавторизованных групп

---

## Быстрый старт

### 1. Клонировать и установить зависимости

```bash
git clone <repo-url>
cd chatgpt-telegram-group-bot
npm install
```

### 2. Настроить переменные окружения

```bash
cp .env.example .env
```

Заполнить `.env` (см. раздел [Переменные окружения](#переменные-окружения)).

### 3. Запустить

```bash
# Сборка и запуск
npm run build
npm start

# Или напрямую через ts-node (для разработки)
npm run dev
```

## Переменные окружения

| Переменная | Обязательная | По умолчанию | Описание |
|---|---|---|---|
| `TELEGRAM_TOKEN` | ✅ | — | Токен бота от [@BotFather](https://t.me/BotFather) |
| `OPENAI_API_KEY` | ✅ | — | API-ключ OpenAI |
| `OPENAI_MODEL` | | `gpt-4o` | Модель ChatGPT (см. ниже) |
| `IMAGE_MODEL` | | `dall-e-3` | Модель для генерации изображений |
| `WHITELIST_USER_IDS` | | — | Разрешённые Telegram user ID через запятую |
| `WHITELIST_GROUP_IDS` | | — | Разрешённые Telegram group ID через запятую |

## Триггерные слова для генерации изображений

Начните сообщение с одного из слов:

- `нарисуй ...`
- `сгенерируй ...`
- `создай картинку ...`
- `создай изображение ...`
- `generate ...`
- `draw ...`
- `image ...`
- `imagine ...`

**Пример:** `нарисуй закат над горами в стиле аниме`

---

## Получение ID пользователя / группы

- **User ID** — напишите боту в личку; если вашего ID нет в whitelist, бот ответит вашим ID.
- **Group ID** — добавьте бота в группу; если группа не в whitelist, бот напишет её ID перед выходом.
